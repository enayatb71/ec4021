##### Make a Python 3 virtual environment and activate it:

On Linux / Mac OS:
```shell
$ python3 -m venv .venv
$ source .venv/bin/activate
```

On Windows:

```shell
$ python3 -m venv .venv
$ .venv\Scripts\activate.bat
```
